## git

git init 创建新的仓库

git clone 复制仓库

git add 添加文件至缓冲区

git rm 删除文件

git commit 确认所作修改

git push 从local到remote 

git pull 根据remote仓库更新local仓库

git merge 将另一个分支的变化合并到本分支
